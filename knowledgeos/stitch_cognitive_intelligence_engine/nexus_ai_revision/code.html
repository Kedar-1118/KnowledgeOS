What is the primary role of backpropagation in training multi-layer perceptrons?

Click to reveal answer

SOLUTION

It calculates the gradient of the loss function with respect to each weight by the chain rule, iterating backward from the last layer to the first. This gradient is then used to update weights via gradient descent.
